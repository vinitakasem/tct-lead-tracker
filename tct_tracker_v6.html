ขั้นตอนการทำงานสำหรับแต่ละ Stage — เจ้าหน้าที่ทำตามนี้ได้เลย

1
Stage 1 — Contacted · หาผู้ติดต่อที่ถูกต้อง
เป้าหมาย: รู้ชื่อและช่องทางของคนที่ถูกต้องก่อนส่ง Pitch
  • Filter Leads ที่มี ⚠️ ไม่มีผู้ติดต่อ — ทำก่อน Tracker
  • ดู Brand Size: Established → ส่ง Email ทางการ / Small → ทัก Line OA Scripts Tab
  • ใช้ Script "Step 1 — หาผู้ติดต่อ" ตามประเภท Brand Size
  • เมื่อได้ชื่อผู้ติดต่อ → อัปเดตใน Tracker ทันที + ตั้ง Follow Up = วันถัดไป
  • บันทึก Note: วันที่ทัก + ช่องทาง + ชื่อที่ได้รับ
2
Stage 2 — Interested · ส่ง Pitch + Catalog
เป้าหมาย: ส่ง Catalog และเปิดบทสนทนาเรื่อง Corporate
  • ใช้ Script "Step 2 — ส่ง Pitch" พร้อมแนบ Catalog link Google Drive
  • ตั้ง Follow Up = 3-5 วันหลังส่ง
  • ถ้าไม่มีการตอบกลับใน 5 วัน → ใช้ Script "Step 3 — Follow Up"
  • บันทึก Note: วันที่ส่ง + ช่องทาง
3
Stage 3 — Qualify · เมื่อ Lead ตอบกลับ
เป้าหมาย: เก็บข้อมูลครบก่อนส่ง Proposal
  • ใช้ Script "Step 4 — Qualify" ถามคำถาม 4 ข้อ
  • บันทึกคำตอบทั้งหมดใน Note ของ Lead
  • ถ้า High Value หรือ Established Brand → แจ้ง Bow ทำ Custom Proposal แจ้ง Bow
  • ตั้ง Follow Up = 2-3 วันหลังได้คำตอบ
P
Stage 3 → 4 — Proposal Sent
เป้าหมาย: ส่ง Proposal และติดตามผล
  • Bow หรือเจ้าหน้าที่อาวุโสจัดทำ Proposal / Quotation Bow ทำ
  • ระบุ Valid Until Date = 14 วัน
  • บันทึกมูลค่า Proposal ใน Tracker
  • Follow Up หลัง 3-5 วัน ถ้ายังไม่มีการตอบกลับ
✓
Stage 4 — Closed Won
เป้าหมาย: Confirm Order + สร้าง Long-term Relationship
  • ใช้ Script "Step 5 — Confirm Order" ยืนยันรายละเอียด
  • ส่งข้อมูลให้ทีม Production ดำเนินการ
  • หลังส่งมอบ: ขอ Feedback / Review
  • ตั้ง Reminder Keep in Touch ทุก 1-2 เดือน Tracker
✕
Stage 5 — Not Interested / On Hold
เป้าหมาย: เก็บไว้ในระบบ + Re-engage ภายหลัง
  • บันทึกเหตุผลใน Note
  • ถ้า "ยังไม่พร้อม" → ตั้ง Re-engage ใน 1-3 เดือน
  • ส่ง Keep in Touch Script ทุก 1-2 เดือน Scripts Tab
